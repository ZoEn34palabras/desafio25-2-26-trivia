# Trivia App - Desafio25-2-26

Aplicación Android desarrollada con Kotlin y Jetpack Compose.

## Funcionalidades

- Quiz interactivo
- 6 preguntas
- Feedback inmediato (Correcto / Incorrecto)
- Botón "Ver resultados" en la última pregunta
- Pantalla final con puntaje

## Tecnologías

- Kotlin
- Jetpack Compose
- ViewModel
- StateFlow
